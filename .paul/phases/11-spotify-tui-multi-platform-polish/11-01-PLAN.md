<objective>
## Goal
Register Spotify tools and the `/spotify` command in the Pi extension, add custom TUI rendering for all Spotify tools, and create extension-level tests — making Spotify a fully functional second platform alongside Apple Music.

## Purpose
This is the final integration phase. After this, users can interact with both Apple Music and Spotify through Pi tools and commands. The LLM can read either library, search either catalog, and create playlists on either platform.

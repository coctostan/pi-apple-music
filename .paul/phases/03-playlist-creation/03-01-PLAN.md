## API Reference
- Create playlist: `POST /v1/me/library/playlists` with body:
  ```json
  {
    "attributes": { "name": "string", "description": "string" },
    "relationships": {
      "tracks": {
        "data": [{ "id": "string", "type": "songs" | "library-songs" }]
      }
    }
  }
  ```
  Response: 201 with created playlist data
- Add tracks: `POST /v1/me/library/playlists/{id}/tracks` with track data array, Response: 204
- Search catalog: `GET /v1/catalog/{storefront}/search?types=songs&term={query}&limit={limit}`
  Response: `{ results: { songs: { data: [{ id, type, attributes: { name, artistName, albumName, ... } }] } } }`
- Storefront: use "us" as default, can be overridden

<objective>
## Goal
Add genre/artist analysis to the library tool and deduplication to playlist creation, giving the LLM richer context for smarter playlist generation.

## Purpose
Currently the LLM sees raw song listings. A genre breakdown and top-artist summary gives the LLM much better signal for recommendations. Deduplication prevents the LLM from adding songs the user already has. Together these make playlist generation significantly smarter.

## Output
- New library action `"genres"` — returns genre frequency breakdown from the user's library
- New library action `"top-artists"` — returns artists sorted by song count
- Playlist creation deduplication — checks trackIds against library and warns about duplicates
- Library song data includes track IDs so the LLM can cross-reference
</objective>

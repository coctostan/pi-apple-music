# Roadmap: pi-apple-music

## Overview
A Pi extension that uses an LLM agent to generate personalized Apple Music playlists by analyzing the user's library and natural language input.

## Phase Details

### Phase 1: Foundation — Scaffold & Apple Music Auth
Scaffold the project from pi-extension-template. Set up Apple Music API authentication (developer JWT token generation + music user token handling). Register the core Pi extension entry point with basic tools and commands.

### Phase 2: Library Reading & LLM Analysis
Implement tools to fetch the user's Apple Music library data (songs, artists, albums, playlists). Structure library data for LLM consumption. Register Pi tools the model can call to read and analyze the library.

### Phase 3: Playlist Generation & Creation
LLM-driven playlist generation from library analysis + user input. Create playlists in the user's Apple Music account via API. End-to-end workflow from natural language prompt to created playlist.

## What Was Done

Two full milestones completed in one session:

**v0.1 Initial Release (3 phases, 3 plans):**
- Scaffolded Pi extension from pi-extension-template
- Apple Music API auth (ES256 JWT developer tokens)
- Config system at ~/.pi-apple-music/config.json
- Library reading: songs, artists, albums, playlists, recently played, summary
- Catalog search for finding songs
- Playlist creation with auto track-type detection
- 3 tools registered: `apple_music_library`, `apple_music_search`, `apple_music_create_playlist`

**v0.2 Smart Playlists & Polish (4 phases, 4 plans, 4 PRs merged via GitHub Flow):**

- Add tracks to existing playlists (`add-tracks` action)
- Genre breakdown and top-artist analysis
- Duplicate detection on playlist creation
- Song listings include track IDs
- Custom TUI rendering (renderCall/renderResult) for all tools
